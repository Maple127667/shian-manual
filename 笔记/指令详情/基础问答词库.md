# 诗岸_official的基础问答词库
## 触发指令
  "山山在吗","诗岸在吗"  
  "山山你好","诗岸你好"  
  "山山可爱","诗岸可爱"  
  "山山骂他","山山骂她","诗岸骂她","诗岸骂他"  
  "山山老婆","诗岸老婆"  
  "山山现在几点了","诗岸现在几点了"  
  "山山今天几号","诗岸今天几号"  
    
  呼叫山山或者诗岸会得到应答~  
  可以和山山说早安晚安~  
  晚安根据时间点不同会有不一样的回复  
## 插件使用平台
  nonebot2 python
